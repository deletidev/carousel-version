## Ejemplo en vivo

- [Ver página](https://studio-d63s.onrender.com/)

## Descripción 📑

Dos ejemplos de carruseles responsive: el primero sólo con un poco de javascript para moverlo con los botones cuando no hay posibilidad de desplazamiento horizontal en el dispositivo, y el segundo tiene un poco más de javascript para hacer cambios basados ​​en el desplazamiento carrusel.

## ¿Qué he aprendido en este proyecto? 🙇🏻

He aprendido a crear carruseles responsive apoyándome en las nuevas propiedades de CSS y con muy poquito de Javascript.
